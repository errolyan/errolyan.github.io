# GitHub 部署指南

## 步骤1：创建GitHub仓库
1. 登录您的GitHub账户
2. 点击右上角的"+"图标，选择"New repository"
3. 为仓库命名（建议使用errol.github.io）
4. 选择公开或私有仓库
5. 不要勾选"Add a README file"、"Add .gitignore"或"Choose a license"
6. 点击"Create repository"

## 步骤2：提供仓库URL
创建完成后，请将仓库URL提供给我，URL格式通常为：
- https://github.com/您的用户名/您的仓库名.git

提供URL后，我将继续帮您完成代码推送和GitHub Pages配置。
